Ebike manager, progetto scolastico relativo alla simulazione di noleggio di EBike Un semplice sistema di simulazione per la gestione del noleggio di biciclette elettriche in un contesto cittadino, sviluppato in C++. Il software permette di localizzare i mezzi più vicini, verificarne l'autonomia e stimare i costi del servizio direttamente da terminale.
